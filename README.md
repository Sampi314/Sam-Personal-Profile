## // About

<p align="center">
  <img src="https://readme-typing-svg.demolab.com?font=JetBrains+Mono&weight=500&size=16&duration=2800&pause=900&color=FF6B00&center=true&vCenter=true&width=560&lines=%3E+what+this+is;%3E+why+it+exists" alt="About typing" />
</p>

A standalone personal website that goes deeper than the GitHub profile - about, work, projects, contact - with full control over branding and tone. Hand-rolled in vanilla HTML/CSS/JS so every pixel earns its place. Treated as v1; canonical successor may be `Sam-Personal-Profile-v2` or `sampi314.github.io`.

## // What's inside

<p align="center">
